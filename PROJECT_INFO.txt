USAGE
-----
From project root directory:

GUI:        python3 gui_distributions.py
Analysis:   python3 main.py -f Distributions/your_file.xlsx

Placement:  python3 main.py -f Distributions/your_file.xlsx --placement cross|x_axis|elliptical
Optimize:   python3 main.py -f Distributions/your_file.xlsx --optimize
            (Alias accepted: --optimise)
            Default seed (if --placement omitted): elliptical
Optimize + seed:
            python3 main.py -f Distributions/your_file.xlsx --optimize --placement elliptical

All Excel files: Distributions/
All figure exports: Figures/

  ✓ 2D Gaussian distribution with arbitrary rotation
  ✓ 2D Pseudo-Voigt as product of 1D functions
  ✓ Custom PSF matrix loading from Excel
  ✓ Rotation-invariant PSF evaluation
  ✓ Automatic normalization to unit integral

main.py:
  ✓ Multi-sheet Excel loading (MM_PSF, Alignment, Gravity, Thermal)
  ✓ Polar grid HEW computation (rotation-invariant)

  ✓ Three runtime modes (coarse/fine/extra-fine)
  ✓ Interactive plotting with keyboard shortcuts
  ✓ Integration with optimizer and placement strategies
  ✓ Automatic timestamped figure export
  ✓ Encircled energy analysis and plotting

gui_distributions.py:
  ✓ Standard preset mode with Excel table integration
