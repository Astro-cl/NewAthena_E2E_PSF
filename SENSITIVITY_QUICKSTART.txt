SENSITIVITY PIPELINE (example)
------------------------------
The repository includes a sensitivity runner that generates per-combo Excel inputs
and can execute them.

Generate per-combo input workbooks only:
  python3 sensitivity/sensitivity_run.py --generate-only --baseline Distributions/Test_Distribution.xlsx

Run the full sensitivity job (generate inputs and execute jobs):
  python3 sensitivity/sensitivity_run.py --baseline Distributions/Test_Distribution.xlsx

Notes:
- Generated input workbooks are written to `sensitivity/input/` (kept, newest 100 preserved).
- Partial per-job results are appended to `sensitivity/results/sensitivity_run_partial.csv`.
- Final consolidated results are written to `sensitivity/results/sensitivity_run_results.xlsx`.
